## Create a site

1. Sidebar → Sites → Create site
2. Name your site; the slug is set automatically
3. Open the site details to continue configuration

## Select spaces

1. Go to Content
2. Add spaces to the Selected list; drag to reorder
3. Use `Add` on each Available space to include it

## Navigation style per space

- Each space appears in the published nav as a dropdown (sidebar layout) or tab (tabs layout)
- Control overall layout in Appearance

<Callout title="Notes" icon="info" color="#10b981">
  You can change selected spaces anytime; publish again to update the live site. The first selected
  space becomes the default landing space.
</Callout>
