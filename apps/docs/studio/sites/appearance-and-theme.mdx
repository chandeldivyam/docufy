## Branding

| **Asset**    | **Tips**                             |
| ------------ | ------------------------------------ |
| Logo (Light) | SVG preferred; for light backgrounds |
| Logo (Dark)  | SVG preferred; for dark backgrounds  |
| Favicon      | .ico or square PNG/SVG               |

## Layout

| **Layout**         | **Best for**                         |
| ------------------ | ------------------------------------ |
| Sidebar + Dropdown | Deep hierarchies and multiple spaces |
| Tabs               | 2–4 spaces of similar importance     |

## Theme Studio

- Fine‑tune color tokens and layout variables
- Token groups: Brand, Surfaces, Sidebar (light/dark), Layout/Scrollbar
- Changes are saved to a site theme; publish to apply them live

<Callout title="Tips" icon="lightbulb" color="#d97706">
  Prefer SVG logos for best scaling; provide both light and dark variants. Set corner radius and
  content padding via layout tokens for a tailored feel.
</Callout>
