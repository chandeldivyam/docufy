## Files to provide

| **Asset**    | **Notes**                                          |
| ------------ | -------------------------------------------------- |
| Logo (Light) | SVG preferred; for light backgrounds               |
| Logo (Dark)  | SVG preferred; for dark backgrounds                |
| Favicon      | .ico or square PNG/SVG; keep small (32–64 px base) |

## Sizing and aspect

- Keep logos readable at small sizes; avoid extremely tall or wide ratios
- Use transparent backgrounds where possible

## Uploads

- Appearance → Branding — drag‑and‑drop or pick files
- You can replace assets at any time; publish to apply changes
