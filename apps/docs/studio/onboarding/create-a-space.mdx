## Steps to create a new space

1. Sidebar → Spaces → Create space
2. Name: visible in navigation (slug auto‑generates)
3. Optional: pick an icon and add a short description
4. Save to create the space and its initial “Getting Started” group

## Space structure

| **Type**           | **Where**      | **Notes**                          |
| ------------------ | -------------- | ---------------------------------- |
| Group              | Root only      | Section header; cannot be nested   |
| Page               | Anywhere       | Rich content in the editor         |
| API Spec           | Anywhere       | Stores OpenAPI; generates children |
| API Tag / Endpoint | Under API Spec | Auto‑generated                     |

<Callout icon="lightbulb" title="Tips" color="#d97706">
  Use multiple spaces when you have distinct audiences (e.g., “User Guide”, “Admin Guide”, “API”).
  Publish one or many spaces to a site and control their order in navigation.
</Callout>
