## Attributes (advanced)

| **Field** | **Purpose**                              |
| --------- | ---------------------------------------- |
| HTML      | Markup rendered in the page              |
| CSS       | Optional styles scoped to this component |
| Display   | block or inline                          |
| Scope     | Optional isolated style region id        |

## Security and safety

<Callout icon="red_circle" title="Important" color="#ef4444">
  Only embed trusted HTML and CSS; do not include scripts. Avoid external iframes unless you trust
  the source.
</Callout>
