## Insert blocks with the slash menu

Type `/` anywhere to open the menu, then search or arrow‑key to select.

| **Block**                      | **Notes**                   |
| ------------------------------ | --------------------------- |
| Heading 1/2/3                  | Semantic sectioning         |
| Text (paragraph)               | Default                     |
| Bullet / Numbered / To‑do list | Lists and checklists        |
| Quote                          | Callouts and quotations     |
| Code block                     | Syntax highlighting         |
| Divider                        | Horizontal rule             |
| Table                          | 3×3 with header by default  |
| Tabs                           | Insert, add, remove, rename |
| Toggle                         | Collapsible section         |

## Inline formatting

- Bold, italics, code, links
- Emoji picker (type `:` to search)

- Insert via `/ Table` → then add/remove rows and columns from the table toolbar
- Columns are resizable; header row is supported
