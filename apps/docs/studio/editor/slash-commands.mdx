Open the slash menu by typing `/` in an empty paragraph, then search or use arrow keys and Enter.

## Commands catalogue

| **Command**           | **What it does**                                |
| --------------------- | ----------------------------------------------- |
| Heading 1/2/3         | Insert a section heading                        |
| Text                  | Paragraph text                                  |
| Bullet List           | Unordered list                                  |
| Numbered List         | Ordered list                                    |
| To‑do List            | Checklist with checkboxes                       |
| Quote                 | Blockquote styling                              |
| Divider               | Horizontal rule                                 |
| Code                  | Code block with syntax highlighting             |
| Table                 | 3×3 table with header row, editable via toolbar |
| Tabs                  | Insert a tabbed content block                   |
| Add/Remove/Rename Tab | Manage tabs inside a tabs block                 |
| Toggle                | Collapsible details/summary section             |
| Custom Component      | Insert custom HTML + CSS block                  |

## Headings and text

- Heading 1 — Big section heading
- Heading 2 — Medium section heading
- Heading 3 — Small section heading
- Text — Regular paragraph

## Lists and tasks

- Bullet List — Unordered list
- Numbered List — Ordered list
- To‑do List — Checklist with checkboxes

## Quoting and structure

- Quote — Blockquote styling
- Divider — Horizontal rule to separate sections

## Code and tables

- Code — Code block with syntax highlighting and Tab indentation
- Table — Inserts a 3×3 table with header row; edit via the floating table toolbar

## Tabs and toggles

- Tabs — Insert a tabbed content block
- Add Tab — Add a new tab to the current tabs block
- Remove Tab — Remove the current tab
- Rename Tab — Rename the current tab
- Toggle — Collapsible section (details/summary)

## Custom HTML

- Custom Component — Insert a custom HTML/CSS block (see Custom HTML Component for guidance)
