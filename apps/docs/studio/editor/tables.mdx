## Editing

| **Action**        | **How**         |
| ----------------- | --------------- |
| Add/Remove row    | Toolbar buttons |
| Add/Remove column | Toolbar buttons |
| Toggle header row | Toolbar toggle  |
| Delete table      | Toolbar action  |

## Resizing

- Drag column borders to resize; widths adjust responsively

<Callout title="Tips" icon="lightbulb" color="#d97706">
  - Use tables for structured data and property lists; avoid very large tables for readability
</Callout>
