## Basics

| **Action**       | **Shortcut**           |
| ---------------- | ---------------------- |
| Open insert menu | `/`                    |
| New paragraph    | `Enter`                |
| Soft line break  | `Shift+Enter`          |
| Code block       | ```` then Space/Enter` |

## Formatting

| **Action**     | **Shortcut** |
| -------------- | ------------ |
| Bold           | `Cmd/Ctrl+B` |
| Italic         | `Cmd/Ctrl+I` |
| Link / Palette | `Cmd/Ctrl+K` |
| Inline code    | `` ` ``      |

## Lists

| **Start**   | **Result**      |
| ----------- | --------------- |
| `- Space`   | Bullet list     |
| `1. Space`  | Numbered list   |
| `[ ] Space` | To‑do list item |

## Navigation

| **Action**         | **Shortcut**                |
| ------------------ | --------------------------- |
| Previous/Next page | `Option+Up` / `Option+Down` |
| Find on page       | `Cmd/Ctrl+F`                |

## Blocks (via /)

- Headings 1–3
- Text (paragraph)
- Bullet / Numbered / To‑do List
- Quote, Divider
- Code block
- Table
- Tabs: Add/Remove/Rename Tab
- Toggle (collapsible)
- Custom HTML Component
